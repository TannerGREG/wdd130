This webpage is a collection of everything I did my first semester at BYU-Idaho. The main project that we had in the class was to make a "White Water Rafting" Page, I ended up making a don't go white water rafting page, because I was bored and it still met all the requirements of the project. We choose this because of the large problem that finding parking is in Rexburg, especailly arround the school. The main component of the website is the map page that uses a google maps API to show the location of the different vehicles arround rexburg.
